Desafio de Código - Conta Bancária em Python

Este é um projeto de um script de conta bancária simples em Python, desenvolvido como parte de um desafio de código para o bootcamp da Digital Innovation One (DIO).

O objetivo deste desafio é criar um programa que simule operações básicas de uma conta bancária, como ver saldo, ver extrato, sacar e depositar dinheiro. O script foi projetado para ser interativo, permitindo ao usuário escolher as ações que deseja executar por meio de um menu.
Funcionalidades

O script oferece as seguintes funcionalidades:

    Ver saldo: Exibe o saldo atual da conta.
    Ver extrato: Mostra o extrato das transações realizadas na conta.
    Sacar: Permite ao usuário realizar saques, desde que haja saldo disponível e o limite de saques não tenha sido excedido.
    Depositar: Permite ao usuário fazer depósitos na conta, desde que o valor seja maior que zero.
    Sair: Encerra a execução do programa.

Execução

Para executar o script, você pode seguir os seguintes passos:

    Certifique-se de ter o Python instalado em sua máquina.
    Faça o download do arquivo conta_bancaria.py deste repositório.
    Abra um terminal ou prompt de comando e navegue até o diretório onde o arquivo foi baixado.
    Execute o seguinte comando: python conta_bancaria.py

O programa exibirá um menu e permitirá que você escolha as opções disponíveis para interagir com a conta bancária.
Observações

    Este é um projeto de exemplo com fins didáticos. Não foi projetado para uso em ambientes de produção ou situações reais.
    O código pode ser expandido e aprimorado para lidar com mais casos de uso, validar entradas do usuário de forma mais abrangente e considerar aspectos de segurança.
    Certifique-se de entender o código antes de executá-lo e faça quaisquer modificações conforme necessário.

Conclusão

Este desafio de código proporcionou uma oportunidade de praticar conceitos de programação em Python, estruturas de controle de fluxo e interação com o usuário por meio de um menu interativo. Ao realizar as operações básicas de uma conta bancária, este projeto serve como um ponto de partida para aprimorar suas habilidades de programação.
